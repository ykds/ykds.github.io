安装 Tomcat 和 Intellij IDEA 开发 Java Web 配置

发表于 2018-02-17 | 分类于 Java | 浏览 次

下载和解压 Tomcat

下载地址：http://tomcat.apache.org/

在页面左边栏可以找到 Download, 按需求选择下载版本，这里选择 Tomcat 9。

下载箭头指的那个，32 位系统就选择上面那个，这是免安装的，解压配置好环境就可以使用。

把下载的压缩包解压，解压位置可自己随意选择。

配置环境变量

Tomcat 解压好后，开始设置环境变量。

配置 Java 的环境变量。

在系统变量中如图添加 JAVA_HOME 变量，变量值改为自己 Java 的安装路径。

然后找到Path变量，添加%JAVA_HOME%/bin;%JAVA_HOME%/jre/bin;到最后，别漏掉了最后的分号。

打开 cmd，分别输入 java 和 javac 出现命令说明就说明配置成功，反之，环境变量配置错误。

配置 Tomcat 环境变量

添加 CATALINA_HOME 变量，值为 Tomcat 的解压路径，然后在Path 中添加 %CATALINA_HOME%/bin;%CATALINA_HOME%/lib; 。

在 cmd 中输入 startup，如果运行结果如图，再访问 localhost:8080, 跳转到 Tomcat 欢迎页面说明安装成功了。

将 Tomcat 安装为系统服务

在 cmd 中运行 service.bat install , 提示安装成功后就能在系统服务找到它。

至此，Tomcat 已经安装完成了。
