蓝桥杯算法题：四平方和

发表于 2017-11-02 | 分类于 算法 | 浏览 次
| 字数统计: 1,203 字 | 阅读时长 ≈ 5 分钟

四平方和定理，又称为拉格朗日定理：
每个正整数都可以表示为至多4个正整数的平方和。
如果把0包括进去，就正好可以表示为4个数的平方和。
比如：
5 = 0^2 + 0^2 + 1^2 + 2^2
7 = 1^2 + 1^2 + 1^2 + 2^2
（^符号表0方的意思）
对于一个给定的正整数，可能存在多种平方和的表示法。
要求你对4个数排序：
0 <= a <= b <= c <= d
并对所有的可能表示法按 a,b,c,d 为联合主键升序排列，最后输出第一个表示法
程序输入为一个正整数N (N<5000000)
要求输出4个非负整数，按从小到大排序，中间用空格分开

例如，输入：
5
则程序应该输出：
0 0 1 2
再例如，输入：
12
则程序应该输出：
0 2 2 2
再例如，输入：
773535
则程序应该输出：
1 1 267 838

根据题目，我们可以得到3个信息：

  1. 对于式子a^2 + b^2 + c^2 + d^2，我们只需要关注系数即可

  2. 系数从小到大排列为 a b c d，后面的每个系数都必须等于或大于前一个系数。如a=2，那么b只能是2、3、4······

  3. 一个整数可能有多种系数序列，但只取众多系数序列升序中的第一个。如整数12，其中两个系数序列是0 2 2 2、1 1 1 3，那么我们只能取0 2 2 2而不能1 1 1 3

    ​

好了，有了这3点信息，我们开始进一步分析，整理思路，编写代码。

假设式子为：a^2 + b^2 + c^2 + d^2

因为我们只需要关注a、b、c、d四个系数，并且系数序列需要的是升序中的首个，那么我们可以用递归的方法，每一个递归层为一个项，那么整个递归就是4层，每层向系数中填入数值，当计算结果大于我们想要的结果时，返回到上一个系数，上一个系数加1后继续往下递归。如此循环直到最后找出可以算出结果的系数。

举个例子：

我们以12作为计结果，那么式子就变成a^2 + b^2 + c^2 + d^2 = 12

从0开始：

0^2 + 0^2 + 0^2 + 0^2 = 0 < 12

0^2 + 0^2 + 0^2 + 1^2 = 1 < 12

0^2 + 0^2 + 0^2 + 2^2 = 4 <12

​ …….

0^2 + 0^2 + 0^2 + 4^2 = 16 >12

当d=4时，计算结果大于12，返回到c，c加1后继续递归，此时d就不能从0开始了，必须从1开始

0^2 + 0^2 + 1^2 + 1^2 = 2 < 12

0^2 + 0^2 + 1^2 + 2^2 = 5 < 12

0^2 + 0^2 + 1^2 + 3^2 = 10 < 12

0^2 + 0^2 + 1^2 + 4^2 = 17 > 12

同是当d=4时结果大于12，重复上面过程

0^2 + 0^2 + 2^2 + 2^2 = 8 < 12

0^2 + 0^2 + 2^2 + 3^2 = 14 > 12

此时，返回c加1后发现：0^2 + 0^2 + 3^2 + 3^2 = 18 > 12， 第一次计算就大于12了，那接下来无论是c还是d都没必要继续增加了，计算结果一定比12大，那么现在就要返回到b，再重复上面的操作

从0^2 + 1^2 + 1^2 + 1^2 = 3 开始

一直到0^2 + 2^2 + 2^2 + 2^2 时，计算结果刚好是12，那么就可以输出这个系数序列0 2 2 2
